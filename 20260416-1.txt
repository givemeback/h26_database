문제 1.
고객별 총 주문금액과 주문 횟수를 보여주는 뷰 v_cust_order_summary를 작성하시오. (출력 컬럼: 고객이름, 총주문금액, 주문횟수)

create view v_cust_order_summary
select c.name,sum(o.saleprice),o.orederid
from customer c
join orders on c.custide = o.custide
group by c.name;



문제 2.
각  도서의  도서명,  출판사,  판매된  총  수량(주문  횟수),  총  판매금액을  보여주는  뷰 v_book_sales를 작성하시오.

create view v_book_sales
select b.bookname,b.publisher,count(o.orderid),sum(o.saleprice)
from Book b
join orders on b.bookid = o.bookid;
group by b.bookname;


문제 3.
주문한 적 있는 고객의 이름, 주소, 가장 최근 주문일을 보여주는 뷰 v_cust_last_order 를 작성하시오.

create view v_cust_last_order
select c.name,c.address,max(o.orderdate)
from customer c
join orders on c.custide = o.custide;
group by c.name,c.address;






문제 4.
도서 정가보다 할인된 금액으로 판매된 주문 내역을 보여주는 뷰 v_discounted_orders를 작성하시오.
(출력 컬럼: 주문번호, 고객이름, 도서명, 정가, 판매가, 할인금액)

create view v_discounted_orders
select o.orderid,c.name,b.bookname,b.price,o.saleprice,(b.price-o.saleprice) as 할인금액
from customer c
join orders o on c.custid = o.custid
join book b on o.bookid = b.bookid;




문제 5.
출판사별  평균  판매가격과  최고  판매가격을  보여주는  뷰  v_publisher_stats를  작성하시 오.
(출력 컬럼: 출판사, 평균판매가, 최고판매가)

create view v_publisher_stats
select b.publisher,avg(o.saleprice),max(o.saleprice)
from book b
join orders o on b.bookid = o.bookid
group by b.publisher;



문제 6.
총   주문금액이   30,000원   이상인   우수   고객의   이름과   총   주문금액을   보여주는   뷰 v_vip_customer를 작성하시오.

create view v_vip_customer
select c.name,o.saleprice
from from customer c
join orders o on c.custid = o.custid
group by c.name
having sum(o.saleprice) >= 30000;



문제 7.
2024년에    주문된    내역의   고객이름,    도서명,    판매가격,    주문일자를   보여주는    뷰 v_orders_2024를 작성하시오.

create view  v_orders_2024
select c.name,b.bookname,o.saleprice,o.orderdate
from customer c
join orders o on c.custid = o.custid
join book b on o.bookid = b.bookid
where TO_CHAR(o.orderdate,'YYYY') = '2024';




문제 8.
한 번도 주문되지 않은 도서의 도서명과 출판사, 정가를 보여주는 뷰 v_unsold_books를 작성하시오.

create view  v_unsold_books
select bookname,publisher,price
from book
where bookid NOT IN (
select bookid
from orders);



문제 9.
고객별로 가장 비싸게 구매한 도서명과 그 금액을 보여주는 뷰 v_cust_max_order를 작 성하시오.
(출력 컬럼: 고객이름, 도서명, 최고구매금액)

create view  v_cust_max_order
select c.name,b.bookname,o.saleprice
from customer c
join orders o on c.custid = o.custid
join book b on o.bookid = b.bookid
where o.saleprice = (
select max(saleprice)
from orders o2
where o2.custid = c.custid);




문제 10.
도서명,  고객이름,  판매가,  해당  도서  평균  판매가,  평균  대비  차이를  보여주는  뷰 v_book_price_compare를 작성하시오.
(출력 컬럼: 도서명, 고객이름, 판매가, 도서평균판매가, 차이)

CREATE VIEW v_book_price_compare AS
SELECT 
    B.bookname AS "도서명", 
    C.name AS "고객이름", 
    O.saleprice AS "판매가", 
    (SELECT AVG(O2.saleprice) 
     FROM Orders O2 
     WHERE O2.bookid = O.bookid) AS "도서평균판매가",
    (O.saleprice - (SELECT AVG(O3.saleprice) 
                    FROM Orders O3 
                    WHERE O3.bookid = O.bookid)) AS "차이"
FROM Customer C
JOIN Orders O ON C.custid = O.custid
JOIN Book B ON O.bookid = B.bookid;
